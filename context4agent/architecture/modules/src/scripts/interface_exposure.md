# md_analysis.scripts — Interface Exposure

## Public API

| Symbol                  | Module        | Description                                              |
|-------------------------|---------------|----------------------------------------------------------|
| `BaderGenError`         | BaderGen.py   | Exception for Bader work directory generation failures   |
| `generate_bader_workdir`| BaderGen.py   | Generate a VASP single-point work directory for Bader charge analysis |

## `generate_bader_workdir` Signature

```python
def generate_bader_workdir(
    atoms: Atoms,
    output_dir: str | Path,
    *,
    script_path: str | Path | None = None,
    workdir_name: str = "bader",
    frame: int = 0,
    source: str = "",
    element_order: tuple[str, ...] | None = None,
    generate_potcar: bool = True,
    direct: bool = True,
) -> Path:
```

**Returns** the created work directory path.

## Generated Directory Contents

| File       | Source                          |
|------------|---------------------------------|
| `POSCAR`   | IndexMapper (element-grouped, with encoded mapping in comment line) |
| `INCAR`    | Template copy (`scripts/template/INCAR`) |
| `KPOINTS`  | Template copy (`scripts/template/KPOINTS`) |
| `POTCAR`   | Generated by `vaspkit 103` (if `generate_potcar=True`) |
| `script.sh`| Copied from `script_path` or persistent config |

## Script Path Resolution

1. If `script_path` is explicitly provided → use it
2. If `script_path is None` → read `KEY_VASP_SCRIPT_PATH` from persistent config
3. If neither → no script.sh copied

## Stability

- **Experimental** — single-frame API; batch multi-frame will be built on top.
- Not re-exported from `md_analysis` top-level `__init__.py`.
